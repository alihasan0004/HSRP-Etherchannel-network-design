# HSRP and EtherChannel Implementation

## Overview

This repository contains my individual contribution to an enterprise network design project developed using Cisco Packet Tracer.

My work focused on implementing:
- HSRP for gateway redundancy and failover
- EtherChannel using LACP for link aggregation
- VLAN trunk verification
- Network redundancy and high availability testing

---

## Technologies Used

- Cisco Packet Tracer
- Cisco IOS
- HSRP
- EtherChannel
- LACP
- VLANs
- OSPF

---

## Project Features

- Active/Standby HSRP configuration
- HSRP failover verification
- EtherChannel link aggregation
- VLAN trunking
- Redundant network paths
- CLI verification and testing
