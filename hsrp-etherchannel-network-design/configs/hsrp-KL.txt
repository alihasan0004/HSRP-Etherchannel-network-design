# HSRP Configuration - KL Branch

## Purpose
Provides gateway redundancy and failover between active and standby multilayer switches for VLAN 10, VLAN 20, and VLAN 30.

## Devices
- KL-SW1 (Active)
- KL-SW2 (Standby)

## VLANs
- VLAN 10: Management
- VLAN 20: HR
- VLAN 30: Design

==================================================
KL-SW1 ACTIVE SWITCH CONFIGURATION
==================================================

interface Vlan1
 no ip address
 shutdown

interface Vlan10
 mac-address 00d0.bc1e.3a01
 ip address 192.168.10.3 255.255.255.0
 standby 10 ip 192.168.10.1
 standby 10 priority 110
 standby 10 preempt

interface Vlan20
 mac-address 00d0.bc1e.3a02
 ip address 192.168.20.3 255.255.255.0
 standby 20 ip 192.168.20.1
 standby 20 priority 110
 standby 20 preempt

interface Vlan30
 mac-address 00d0.bc1e.3a03
 ip address 192.168.30.3 255.255.255.0
 standby 30 ip 192.168.30.1
 standby 30 priority 110
 standby 30 preempt

==================================================
FAILOVER TEST
==================================================

Failover testing was performed by shutting down the active HSRP interface on KL-SW1.

The standby multilayer switch (KL-SW2) successfully transitioned into the active state without requiring any changes to end-device default gateway configuration.

This verified successful HSRP redundancy and high availability operation across VLAN 10, VLAN 20, and VLAN 30.
